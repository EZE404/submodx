Privacidad y funcionamiento
  • Clave personal: Cada usuario usa su propia clave API de SubX. Tu clave solo se usa para realizar tus solicitudes de subtítulos (búsqueda y potenciales descargas).
  • Límites de uso: SubX tiene límites de solicitudes por clave API (bastante generosos). En uso personal normal este límite no es alcanzable. SubmodX utiliza caché para prevenir solictudes repetidas inmediatas a SubX.

    Si no hay datos en caché y se han agotado las solicitudes a SubX de tu clave API, SubmodX no devolverá resultados hasta que tu cuota se renueve, o que otro usuario haya realizado esa misma petición (y por tanto, haya quedado en caché).

    Actualmente, la caché de resultados de búsqueda no vuelve a consultar SubX antes de {{SEARCH_CACHE_TTL}} desde la última consulta. El tiempo es corto ya que podrías estar esperando los subtítulos para un estreno. Mientras que la caché para un archivo de subtítulos es de {{DOWNLOAD_CACHE_TTL}}, por lo que no deberías preocuparte por tu cuota cuando seleccionas nuevamente un mismo archivo de subtítulos.

    La cantidad máxima de subtítulos solicitados a SubX por búsqueda (película o episodio) es de {{MAX_SEARCH_RESULTS}}.

    Para conocer los límites exactos de SubX consulta
    subx-api.duckdns.org/docs/es/guides/rate-limits/.
  • Privacidad total: Tu clave viaja cifrada en la URL personal generada para instalar el complemento. El servidor la descifra al vuelo para cada petición. Así que, no compartas la URL generada. Comparte esta página para que cada usuario genere su propia URL.

    No se guarda nada, ni en base de datos ni en disco. No necesitas pedir que se borren tus datos porque no se almacenan.
